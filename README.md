# 技術書籍感想文

[AkiUnleash/ book_Impressions_back](https://github.com/AkiUnleash/book_Impressions_back) にてGoで記載したREST APIをScalaで書き直したリポジトリ。


## 説明

・REST API にて JSON をレスポンスするプログラムです。  
・JWT 認証を使用しています。（サーバー側から操作し Cookie に保存。)

## システム・アーキテクチャ

- Scala
- MySQL
- Akka HTTP
- Slick
- Spring
